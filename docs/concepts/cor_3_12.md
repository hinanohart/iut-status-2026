# [iut:Cor.3.12] Corollary 3.12 (the disputed corollary) — Mochizuki view
> 3-agent verify pending (mochizuki-side draft).

## Statement (verbatim quotation, IUTchIII Cor.3.12)

IUTchIII Corollary 3.12 は IUTchIII Theorem 3.11 (multiradial representation) の
直接の帰結として述べられる log-volume 不等式である。
Mochizuki 自身による最粗い要約 (Rpt2018 §2, label (Smm)) は以下の通り:

> Suppose that A and B are positive real numbers, which are defined so as to satisfy
> the relation −2B = −A (which corresponds to the Θ-link). One then proves a theorem
> −2B ≤ −2A + 1 (which corresponds to the multiradial representation of [IUTchIII],
> Theorem 3.11). This theorem, together with the above defining relation, implies a
> bound on A: −A ≤ −2A + 1, i.e., A ≤ 1
> (which corresponds to [IUTchIII], Corollary 3.12).

Statement-level の数式表記: log-volume 不等式

```
−|log(Θ)| ≥ −|log(q)|  (大雑把な対応; 厳密版は IUTchIII p.173–174)
```

実際の Corollary は「Θ-pilot の log-volume が q-pilot の log-volume 以上」を
indeterminacy (Ind1, Ind2, Ind3) を込みで確立する。
これが Section 7 の abc / Szpiro 不等式への直接の橋渡しとなる。

- Reference: IUTchIII Cor.3.12, pp.173–174, DOI [10.4171/PRIMS/57-1-3](https://doi.org/10.4171/PRIMS/57-1-3)

## Mathematical content

### Theorem 3.11 (multiradial representation) → Cor.3.12 の流れ

1. **log-theta-lattice の構成**
   - 水平方向: Θ-link が (0-column) の Θ-holomorphic structure と
     (1-column) の q-holomorphic structure を poly-isomorphism でつなぐ。
   - 垂直方向: log-link が各 column 内の ring structure を反復適用される
     p-adic 対数写像で結ぶ (Frobenius-like 構造)。
   - Θ-link は ring homomorphism **ではない** (ΘNR)。
     これが二つの holomorphic structure を「別物」として保持する必要性の根拠。

2. **Indeterminacy の役割 (Ind1 / Ind2 / Ind3)**
   - (Ind1): Θ-link の構成に現れる G-action の poly-morphism 由来。
   - (Ind2): O×μ を abstract topological monoid として扱う際の不定性。
   - (Ind3): log-link の反復で生じる additive 構造の不定性
     (log-Kummer-correspondence の kernel)。
   - これら三つを明示的に許容することで、Θ-pilot の multiradial 表現が
     「0-column と 1-column の switching に対して compatible」になる (SW 性質)。
   - Mochizuki (Rpt2018 §8) は (Ind1, Ind2) を「joint の柔軟性」に例え、
     剛体に switching 対称性を持たせるための必須コストと論じる。

3. **Frobenius-picture と log-Kummer-correspondence**
   - Frobenius-picture: HT^Θ chain を横方向に並べた図式。
     各 Hodge theater の Frobenius-like 構造が Θ-link で接続される。
   - log-Kummer-correspondence: 0-column の étale-like 構造が
     log-link の反復に対して「invariant (up to Ind3)」であることを保証する。
     これが Θ-pilot の log-volume を 1-column の q-pilot と比較可能にする鍵。

4. **log-volume 積分と Cor.3.12 の核**
   - EssLgc §3.10 の (Stp1)–(Stp8) に詳述される「8 ステップ」:
     - (Stp1): Ind3 を 0-column に局所化。
     - (Stp2)–(Stp3): multiradial 表現の descent、holomorphic hull の構成。
     - (Stp4)–(Stp6): hull-approximant への passage、行列式冪の調整。
     - (Stp7): log-shift 補正 (1-column の (1,m)→(1,m+1) 対角矢印の symmetry)。
     - (Stp8): log-volume を取ることで「closed loop」が閉じ、
       distinct label が消去され、nontrivial set-theoretic 不等式が得られる。
   - この最終段階の不等式が Cor.3.12 の statement である。
   - EssLgc §3.10 (LVsQ): 「log-volume を取ることで初めて
     formal subquotient が coarse/set-theoretic subquotient になる」。

## Mochizuki 2018 response to Scholze-Stix

### SS 主張の核心 (Mochizuki による要約)

Rpt2018 §7 の (SSInd):
> The essential content (e.g., multiradial algorithms) of IUTch is entirely
> unaffected even if the indeterminacies (Ind1, 2) are eliminated.

SS は「Θ-link の domain と codomain の ring structure を canonical に同一視できる
(SSId)」と主張し、その場合に A = B = 0 という矛盾が生じると論じた (SS 矛盾論法)。

### Mochizuki の反論 (5 点)

1. **「SS が作った矛盾は IUTch の矛盾ではない」** (SSADFs, Rpt2018 §12)
   - 矛盾 (AD) は SS が ring structure を両側で同一視した **改変版 IUTch** に
     内在する矛盾に過ぎない。元の IUTch は distinct label を維持するため矛盾しない。
   - Rpt2018 (SSADFs): 「The assertion of (SSAD) is false. The contradiction only
     occurs when one identifies the hol. strs./ring theories in the domain and
     codomain of the Θ-link.」

2. **「distinct label の削除は ring structure の混同と同値」** (SSDLFs, Rpt2018 §12)
   - label を消去することは (Ind1, Ind2) を消去するより **さらに強い** 操作であり、
     multiradial algorithm が適用不能になる。

3. **「SS の §2.2 議論は proof of Cor.3.12 の描写ではない」** (Cmt2018-05 (C17), (C18))
   - Cmt2018-05 (C17) は SS 原稿 §2.2 の議論について:
     「log-volume を計算する際に commute が必要な 'loop' は
     Cor.3.12 の proof には **決して現れない**」と断言。
   - (N1)–(N5): 「copies of R の canonical identification」も
     「displayed diagram の commutative loop」も証明には存在しない。

4. **「multiradial 表現の common container が loop 問題を解消する」** (Cmt2018-05 (C18))
   - Θ-pilot と q-pilot は **単一の container** に (indeterminacy を許して)
     埋め込まれる。container は一つなので log-volume 関数は well-defined であり、
     「二つの distinct mapping の commutativity 問題」は発生しない。

5. **「SS は Theorem 3.11 の validity 自体は争っていない」** (IUT2025-10 §2.1, SSA1)
   - 2025 年時点でも Mochizuki は確認: SS は Theorem 3.11 (multiradial algorithm) の
     数学的正当性を否定しておらず、主張の核は (SSA1):
     「[IUTchI-IV] から nontrivial な diophantine 不等式を導く方法が見えない」
     という理解不足に留まると論じる。

## Mochizuki 2025-10 update

**出典: IUT-report-2025-10.pdf (October 2025)**

1. **FA1 の否定**: 「log-theta-lattice に intrinsic な矛盾はない」を改めて明確化。
   Boyd [Ctv] 論文が主張する「IUT は labels によって矛盾を宙吊りにする」という
   叙述 (FA2) も完全に誤りと断じる。
   - IUT2025-10 §2.1 (SSA2): SS の argument は
     「Cor.3.12 の set-up の SS 流解釈/simplification」であり、
     実際の log-theta-lattice の setup および Cor.3.12 の論証と
     「論理的な関係が存在しない (no logical relationship)」。
   - Existence Principle (ExPr): 「nontrivial な帰結を出せない algorithm
     (SS simplification) の存在は、帰結を出す algorithm (Cor.3.12) の
     非存在を含意しない」という形式論理を強調。

2. **Lean4 形式化の戦略的重要性**
   - IUT2025-10 §3.2: 2025 年 7 月東京 Lean workshop にて、
     SGA1 の profinite fundamental group の Lean4 形式化が完成したことが報告された。
     Mochizuki はこれを根拠に「少なくとも理論的には IUT の Lean4 形式化に
     fundamental な障害はない (NoObs)」と述べる。
   - 進行中の課題: Hodge theater 構成と Theorem 3.11 の形式化には
     数年規模の work が必要 (anabelian geometry の blackbox 化が鍵)。
   - IUT Summit 2028 (spring 2028 予定) が次の主要 workshop として計画中。
   - EssLgc §3 の ∧(∨)-chain (Boolean AND/XOR の連鎖) を
     「IUT の essential logical structure を純粋に combinatorial な言語で表現する
     Lean4 形式化への橋渡し」と位置付ける (MthFm, BBxFm)。

3. **形式化への一次障害は数学的内容ではなく人材**
   - IUT の rigorous な理解と Lean コード執筆能力を兼ね備えた数学者が
     現時点では存在しない (LnCom) ことが実務上の最大障害。

## Why Cor.3.12 is the load-bearing step

- IUTchIII の論証構造において、Theorem 3.11 は multiradial algorithm の
  **存在と switching compatibility** を保証するが、
  Cor.3.12 が初めて「log-volume の具体的不等式」として結果を固定する。
- この不等式が IUTchIV Theorem 1.10 を経由して
  **Szpiro 不等式 / abc 予想の effective height bound** へ直結する
  (EssLgc (vi-k); Rpt2018 (BgIUT2))。
- Rpt2018 §15 の Bogomolov 類比:
  displacement estimates (Bogomolov) ↔ Theorem 3.11;
  height computation via q-parameters (Bogomolov) ↔ **Cor.3.12**.
- Section 7 へのリンク: `docs/concepts/section_7_abc_consequence.md` (未作成)
